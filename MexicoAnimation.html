En este ejemplo vemos un cursor animado tipo introducción de texto en terminal
Este efecto está en la hoja de estilo copiada del sitio web de kodemia.mx


Concretamente la parte del código que define este cursor parpadeando es la siguiente:

Dentro del título H3 la letra "s" esta rodeada de la etiqueta span:


        <span class="border-b-4 border-solid animate-cursor text-brand-primary">s</span>
Partes del código CSS:

        .border-b-4{border-bottom-width:4px}
        .border-solid{border-style:solid}
        .animate-cursor{-webkit-animation:cursor .8s linear infinite alternate;animation:cursor .8s linear infinite alternate}
        .text-brand-primary{--tw-text-opacity:1;color:rgb(255 155 55/var(--tw-text-opacity))}
        .text-brand-complementary{--tw-text-opacity:1;color:rgb(255 155 251/var(--tw-text-opacity))}
        .hover\:text-brand-primary:hover{--tw-text-opacity:1;color:rgb(0 255 255/var(--tw-text-opacity))}

Primeramente incluí la hoja de estilos original - mexicoStyle.css
Después manualmente limpié lo que pude obteniendo mexicoLindo.css
Aún así, me parecía que había demasiado código dentro así que busqué una forma de limpiarlo automática.
https://purifycss.online/ fué la solución, resultando en un archivo mucho más ligero - mexicoReLindo.css aunque se ha vuelto a poner todo en una línea. Para su mejor lectura debería añadir intros...
